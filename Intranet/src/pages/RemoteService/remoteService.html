Atenção

Para realizar um atendimento a distância, tenha o IP do usuário.

Passo 1

Obter o endereço da máquina(IP), o usuário pode não saber, o obtenha via TightVNC passando o mouse por cima do icone do aplicativo ou no CMD com ipconfig

Passo 2

Endereço coletado, agora abra o TightVNC, insira o IP e a senha

BÔNUS - Instalação de Software

Passo 1

Entrar no IP de servidor do repositório de programas

IP: \\192.168.0.153\d

ou: \\192.168.5.153\d

Passo 2

Escolher a pasta "/Softwares/" e instalar o software desejado.
